This is an implementation of the paper "Music genre classification using a hierarchical long short term memory (LSTM) model" by Tang et al.

The provided code is a colab notebook file. Simply run it from start to finish on colab.

The GTZAN dataset is downloaded straight from torchaudio in the provided code.

! To download the dataset for the first time, comment out the parts copying from google drive. !
After dowloading the dataset once, you can uncomment the code that transfers the dataset to/back from drive
